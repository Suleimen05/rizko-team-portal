7. Чеклист реализации Прогресс
  • ✓ pgvector — расширение включено в Supabase
  • ✓ Embeddings таблица — 64 записи с HNSW индексом
  • ✓ Voyage AI — подключён, генерирует embeddings
  • ✓ Chunking — шаблоны для 6 типов данных (contacts, tasks, calendar, scripts, wiki, strategies)
  • ✓ Semantic Search — cosine similarity через search_embeddings()
  • ✓ Re-ranking — keyword + vector hybrid scoring
  • ✓ RAG Chat — сервер автоматически ищет и добавляет контекст
  • ✓ Batch Sync — POST /api/rag/sync для полной синхронизации
  • ✓ Single Record Sync — POST /api/rag/sync-record для обновления одной записи
  • ✓ RAG Logs — метрики каждого запроса записываются
  • ✓ Context Stuffing удалён — фронтенд больше не грузит все данные
  • ✗ Auto-embed — автоматический sync после CRUD операций
  • ✗ Semantic Cache — кэширование похожих запросов
  • ✗ Function Calling — AI сам выбирает какой SQL запрос делать
  • ✗ Cohere Re-rank — cross-encoder вместо keyword overlap
  • ✗ LangSmith/Langfuse — трейсинг и мониторинг

Статус: 11 из 16 пунктов реализовано (69%). Основной RAG pipeline полностью работает. Оставшиеся 5 пунктов — оптимизация и продвинутые фичи для масштабирования.
